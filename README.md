 Jeu de Devinettes & Énigmes

 # Description

Ce projet est une application web interactive proposant un ensemble de (devinettes, charades et énigmes) destinées à stimuler la réflexion et la logique des utilisateurs.
L’objectif principal est de résoudre chaque défi en faisant appel à l’observation, à la déduction et à la créativité.

 # Objectifs

- Offrir une expérience ludique et éducative
- Stimuler les capacités cognitives (logique, mémoire, réflexion)
- Proposer un contenu accessible à tous (débutants comme confirmés)

# Fonctionnalités

- Affichage de devinettes et d’énigmes variées
-  Saisie de réponses par l’utilisateur
-  Vérification des réponses
-  Possibilité d’afficher la solution
-  Navigation entre différentes questions

# Technologies utilisées

- *HTML* : structure de la page
-  *CSS* : mise en forme et design
-  *JavaScript* : logique du jeu et interactions

#  Utilisation

1. Ouvrir le jeu dans un navigateur web
2. Lire attentivement la devinette affichée
3. Proposer une réponse
4. Vérifier si la réponse est correcte ou afficher la solution
